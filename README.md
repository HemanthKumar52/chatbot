# BGS International School Chatbot

AI-powered chatbot for BGS International School information.

## Quick Start - Choose ONE method:

### Method 1: Direct Browser (Easiest - Works Now!)
1. Double-click `index.html`
2. Or right-click → Open with → Chrome/Firefox/Edge

### Method 2: With Node.js Server (For API Features)
```bash
# Install Node.js from https://nodejs.org/ (if not installed)

# Then run:
node server.js

# Open browser: http://localhost:8000
```

### Method 3: VS Code Live Server
1. Install "Live Server" extension in VS Code
2. Right-click `index.html`
3. Select "Open with Live Server"

## Features
- ✅ Answer BGS-specific questions
- ✅ Leadership information (Founder, Chairman, Principal, etc.)
- ✅ School hours, contact, admissions
- ✅ Fee structure and policies
- ✅ Voice input support
- ✅ Quick question buttons

## Important Files
- `index.html` - Main page
- `style.css` - Styling
- `script.js` - Chatbot logic
- `.env` - Configuration (API keys)
- `server.js` - Optional Node.js server
